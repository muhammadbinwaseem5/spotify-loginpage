# Spotify Login Page Clone

A pixel-accurate clone of the Spotify login page, built from scratch using HTML and CSS.

**[Live Preview](https://muhammadbinwaseem5.github.io/spotify-loginpage/)**

## Built With

- **HTML5** — semantic structure and form elements
- **CSS3** — layout, styling, and hover interactions
- **Font Awesome 6.5** — Google, Facebook, and Apple icons inside social buttons
- **Google Fonts** — Google Sans for typography


## Features

- Spotify logo centered at the top
- Email input field with focus border effect
- Green branded Continue button with hover scale effect
- Social login buttons (Google, Facebook, Apple) with icons pinned to the left
- Hover effects on all buttons using CSS transforms
- "Don't have an account? Sign up" link at the bottom
- Fully centered layout using CSS Flexbox
